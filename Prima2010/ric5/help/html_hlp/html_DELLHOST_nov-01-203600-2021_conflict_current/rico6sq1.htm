Altre funzioni

Con il pulsante pulisci si ottiene la cancellazione di quanto compare sia nella lista che nella finestra di modifica-immissione. Tale cancellazione non è permanente, cioè i dati rimangono memorizzati. Per cancellarli in modo definitivo occorre premere il pulsante elimina dopo aver portato il dato, con un doppio click, dalla finestra di lista a quella di immissione.

Con i pulsantini 1 - 2 - 3 - 4 vengono resi operativi uno dei quattro file disponibili per Ricorda 32.

Il pulsante elenco lista tutti i record contenuti nel file. Tali records possono essere anche stampati.

Il pulsante stampa effettua la stampa dei dati presenti nella lista.

All’avvio del programma, il sistema effettua un ciclo di ordinamento dei dati ed anche un ciclo di attualizzazione. Tale operazione avviene all’atto della selezione del file da utilizzare, e quindi al lancio del programma, per il primo dei files che viene considerato di default.

In modo automatico, Ricorda realizza anche una copia di riserva dei vostri dati, per proteggerli da eventuali errori. Questa copia avviene all'atto del lancio del programma per il file di default, Ricorda1.dat e quando viene premuto il relativo pulsantino di attivazione ( 1, 2, 3, 4), per gli altri files .dat . Il file di riserva ha lo stesso nome di quello principale, ma con una estensione .dak.

In caso di necessità è possibile cancellare il file .dat e rinominare quello di riserva cambiando l'estesione da .dak a .dat. In tal modo si otterrà un file dati relativo alla situazione dei dati prima dell'utilizzo corrente di ricorda.
